# Snake Water Gun Game

A modern web-based implementation of the classic Snake, Water, Gun game (similar to Rock-Paper-Scissors). This project allows a user to play against a computer with an interactive interface, animations, and score tracking.

---

## Features

* Single-player mode against computer AI
* Randomized computer opponent names
* Real-time score tracking (first to 5 points wins)
* Smooth UI transitions and animations
* Separate screens for menu, gameplay, and results
* Responsive design for different screen sizes

---

## Technologies Used

* HTML5 – Structure of the application 
* CSS3 – Styling, layout, and animations 
* JavaScript – Game logic and interactivity 

---

## Game Rules

* Snake defeats Water
* Water defeats Gun
* Gun defeats Snake
* Same choices result in a draw

The first player to reach 5 points wins the match.

## How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/snake-water-gun.git
   ```

2. Open the project folder

3. Run the application by opening `index.html` in any web browser

---

## Working Principle

* The player enters their name and starts the game
* The computer randomly selects a choice
* The game logic determines the winner of each round
* Scores are updated dynamically
* The match ends when one player reaches 5 points
